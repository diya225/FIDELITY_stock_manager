# Stock Manager Website - Feature Overview

## What This Website Is

Stock Manager is a beginner-friendly virtual stock portfolio website for people who want to understand investing before putting real money into the market.

The website helps users create a financial profile, discover suitable stock recommendations, buy and sell stocks virtually, and track how their portfolio is performing through a simple dashboard.

It is especially useful for new investors because it does not just show stock names. It also explains why a stock may be suitable, how much money could be invested, and how the user’s portfolio is balanced.

## Who Can Use It

This website is designed for:

- First-time investors who want a safe place to learn.
- Students or young professionals who want to understand stock investing.
- Retail investors who want simple recommendations based on their financial situation.
- Users who want to practice trading without using real money.
- Admins who manage users, stocks, and stock price information.

## Main Things Users Can Do

## 1. Create an Account

Users can register with their name, email, and password.

After creating an account, they can log in securely and access their personal dashboard. The website protects accounts using password validation and login security.

## 2. Complete a Financial Profile

The website asks the user for basic financial details such as:

- Monthly income
- Monthly expenses
- Current savings
- Investment goal
- Risk appetite

The profile is shown as a step-by-step wizard, so users do not have to fill everything at once on a confusing long form.

## 3. Calculate Investable Amount

Based on the financial profile, the website calculates how much money the user can safely use for virtual investing.

For example, it considers:

- How much money is left after monthly expenses
- Whether the user is conservative, moderate, or aggressive
- Whether the goal is short-term, medium-term, or long-term

This helps users understand that investing should be based on available surplus money, not random guesses.

## 4. Get Stock Recommendations

After the profile is completed, the website can generate stock recommendations.

Each recommendation shows:

- Stock ticker
- Company name
- BUY, HOLD, or SELL signal
- Suitability score
- Confidence level
- Suggested investment amount
- Current price
- Sector
- A simple explanation

The goal is to help users understand not only what stock is recommended, but also why it is recommended.

## 5. Search and Filter Recommendations

Users can browse recommendations more easily using:

- Search by company name or ticker
- Filter by signal type such as BUY, HOLD, or SELL

This makes it easier to find a specific company or focus only on buying opportunities.

## 6. Buy Stocks Virtually

Users can buy recommended stocks using virtual money.

The website checks important rules before allowing a buy:

- The user must have enough virtual balance
- The stock must have price data
- Quantity must be a whole number
- The trade cannot go beyond the user’s investable amount

This gives users a realistic paper-trading experience without financial risk.

## 7. Sell Stocks Virtually

Users can sell stocks from their portfolio.

When selling, the website:

- Checks how many shares the user owns
- Prevents selling more than the user has
- Calculates trade value
- Calculates realized profit or loss
- Adds the money back to virtual balance

This helps users learn how profits and losses work in a portfolio.

## 8. View Portfolio Holdings

The portfolio page shows all stocks currently owned by the user.

Users can see:

- Stock name and ticker
- Quantity owned
- Average buy price
- Current price
- Profit or loss
- Available cash balance
- Total market value

This gives users a clear view of what they own and how each holding is performing.

## 9. Dashboard Summary

The dashboard gives a quick overview of the user’s investment position.

It includes:

- Total portfolio value
- Predicted profit or loss
- Number of holdings
- Portfolio score
- Remaining virtual balance

This is the main place where users can quickly understand their portfolio health.

## 10. Portfolio Score

The website gives the portfolio a score out of 100.

The score is based on factors such as:

- Diversification
- Risk
- Profitability
- Volatility
- Consistency

This makes portfolio quality easier to understand for beginners.

## 11. Charts and Visual Insights

The dashboard includes visual charts so users can understand their portfolio faster.

Charts include:

- Portfolio value over time
- Sector allocation
- Gainers
- Losers

Instead of reading only numbers, users can visually see where their money is invested and how it is moving.

## 12. Sector Allocation

Users can see how their portfolio is divided across sectors such as:

- IT
- Banking
- Energy
- FMCG
- Pharma
- Automobile
- Metals
- Infrastructure

This helps users avoid putting all their money into one type of company.

## 13. Gainers and Losers

The dashboard shows which stocks are performing best and worst.

This helps users quickly identify:

- Which stocks are gaining value
- Which stocks are losing value
- Which holdings may need attention

## 14. AI-Assisted Recommendation Engine

The project includes a recommendation service that scores stocks and suggests whether they are suitable for the user.

The recommendation engine considers:

- Stock movement
- Sector
- User risk profile
- Investable amount
- Suitability score

The explanations are written in simple language so users can understand the reason behind each recommendation.

The project includes sample Indian stock data and demo users.

This means users can test the website without manually adding everything from scratch.

## 16. Admin Features

Admins can manage important platform information.

Admin functionality includes:

- Viewing users
- Updating stock prices
- Activating or deactivating stocks

This helps platform operators maintain the website data.

## 17. Safe Learning Environment

The website is virtual, so users can learn investing without using real money.

This is useful because beginners can:

- Practice buying and selling
- Learn portfolio tracking
- Understand investment risk
- See how diversification works
- Make mistakes safely

## 19. Mobile-Friendly Website

The website is designed to work on different screen sizes, including:

- Mobile phones
- Tablets
- Laptops
- Desktop screens

Users should be able to check recommendations and portfolio details even from a smaller device.

## 20. What Users Get From This Website

By using this website, a user can:

- Understand how much they may be able to invest.
- Learn how risk appetite affects investment decisions.
- Receive stock recommendations based on their profile.
- Practice virtual buying and selling.
- Track profit and loss.
- Understand portfolio diversification.
- View their financial decisions visually through charts.
- Build confidence before investing real money.

## In Simple Words

This website acts like a guided stock investing practice platform.

It helps users answer questions like:

- How much can I invest?
- Which stocks may suit me?
- How risky is my portfolio?
- Am I diversified enough?
- Am I making profit or loss?
- What should I watch next?

The main purpose is to make stock investing easier to understand, safer to practice, and more approachable for beginners.
